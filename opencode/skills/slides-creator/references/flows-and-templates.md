# Flows and Templates (Markdown Slides)

Use this file only when you need a quick "menu" of deck flows or slide skeletons.

## Flow menu (pick 1)

- Proposal / decision
  - Context -> problem -> options -> recommendation -> plan -> risks -> ask
- Technical walkthrough
  - Problem -> approach -> system -> key details -> eval -> limits -> next
- Research / paper
  - Question -> method -> setup -> results -> ablations -> takeaways
- Status update
  - Goal -> last period -> metrics -> wins -> issues -> next period -> asks
- Pitch
  - Why now -> problem -> solution -> proof -> plan -> ask

## Slide skeletons (copy + fill)

### Title

```md
# Topic (short)
- One-liner (what + why)
  - Audience / use-case
    - Date / presenter (opt)
```

### Agenda / flow

```md
# Agenda
- Section 1 (verb phrase)
  - Subpoint (scope)
    - Why it matters
- Section 2 ...
  - ...
    - ...
```

### Problem

```md
# Problem
- Symptom (what hurts)
  - Who feels it
    - How often / how big
- Root cause (why)
  - Constraint / bottleneck
    - Evidence (1 line)
```

### Approach / solution

```md
# Approach
- Core idea (1 line)
  - Key mechanism
    - Why it works
- Design choices (2-3)
  - Trade-off
    - Rationale
```

### Options + recommendation

```md
# Options
- A: ...
  - Pros
    - ...
  - Cons
    - ...
- B: ...
  - ...
    - ...
```

```md
# Recommendation
- Choose: X
  - Why now
    - Expected impact
- Guardrails
  - Risk + mitigation
    - Owner / next step
```

### Results / metrics

```md
# Results
- Metric 1: before -> after
  - Method / caveat
    - Link / source (opt)
- Metric 2: ...
  - ...
    - ...
```

### Risks

```md
# Risks
- Risk 1
  - Impact
    - Mitigation
- Risk 2
  - ...
    - ...
```

### Next steps / ask

```md
# Next
- Decision needed
  - Options
    - Recommendation
- Actions (who/when)
  - Owner: ...
    - ETA: ...
```
